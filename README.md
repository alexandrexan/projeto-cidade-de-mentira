# Cidade de Mentira - Projeto de HTML/CSS

Projeto estático inspirado no desafio 012 do Curso em Vídeo. O foco aqui é **composição visual, tipografia expressiva e ritmo de leitura**, criando uma página de poesia com presença forte já no primeiro viewport.

## Destaques
- Hero full-bleed com imagem dominante e contraste controlado.
- Tipografia com personalidade e hierarquia clara.
- Layout responsivo com escalas fluidas (`clamp`).
- Efeitos de profundidade e parallax em telas maiores.
- Animações sutis de entrada e foco em legibilidade.

## Tecnologias
- HTML5
- CSS3 (sem frameworks)

## Como executar
1. Abra o arquivo `index.html` no navegador.

## Estrutura do projeto
- `index.html` - marcação principal.
- `style.css` - estilos e animações.
- `imagens/` - assets visuais.

## Objetivo
Demonstrar domínio de **layout, tipografia e narrativa visual** usando apenas HTML/CSS, com atenção a contraste, ritmo e experiencia mobile.

## Créditos
Letra e referência artística: Subsolo e Gato Congelado.
