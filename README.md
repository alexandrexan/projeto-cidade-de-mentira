# Cidade de Mentira



![preview do projeto](./assets/preview.png)

## Visão geral
Página estática construída para explorar **tipografia como elemento central de interface**, com foco em leitura, contraste e impacto visual imediato. A estrutura prioriza hierarquia clara e ritmo, conduzindo o usuário ao longo do conteúdo sem dependência de scripts.

## Decisões técnicas
- **Tipografia como layout**: uso de escalas fluidas com `clamp()` para manter proporção entre títulos e corpo em diferentes telas.
- **Contraste controlado**: sobreposição de imagem com camadas de cor para garantir legibilidade sem perder impacto visual.
- **Responsividade mobile-first**: adaptação progressiva com breakpoints mínimos, priorizando leitura em telas pequenas.
- **Profundidade visual**: efeitos de parallax aplicados apenas em telas maiores para evitar impacto em performance mobile.
- **Animações sutis**: transições leves para reforçar foco e leitura, sem interferir na experiência.

## Tecnologias
- HTML5 semântico  
- CSS3 puro (sem frameworks)

## Estrutura
- `index.html` — estrutura da página  
- `style.css` — estilos, responsividade e animações  
- `imagens/` — assets visuais  

## Execução
Abra o arquivo `index.html` em qualquer navegador moderno.

## Foco técnico
Implementação orientada a **UI/UX com base em HTML e CSS**, priorizando:
- legibilidade
- hierarquia visual
- consistência responsiva
- performance sem dependências externas
